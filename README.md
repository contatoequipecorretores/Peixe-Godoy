# Peixe Godoy

Site estático em React + TypeScript configurado com Vite. Preparado para deploy no Netlify.

Como desenvolver:

```bash
npm install
npm run dev
```

Build para produção:

```bash
npm run build
npm run preview
```

Deploy no Netlify:

- Conectar repositório ao Netlify
- Build command: `npm run build`
- Publish directory: `dist`
